### Hi, there 🙌

이 리포지터리는 카프카 관련된 개념들을 정리했던 문서들을 보관하는 리포지터리입니다. 1년 가까이 시간이 날때마다 정리해왔던 내용들 중 필요한 파일들만 추려두었고, 앞으로도 추가적으로 자료들을 추가해나가거나, 부정확한 내용들은 수정해나갈 예정입니다.<br>

보여지기 위해 운영하는 리포지터리가 아니다보니, 가끔가다 부정확한 내용이 보이더라도 양해부탁드립니다.<br>
